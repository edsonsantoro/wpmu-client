== Description ==

This plugin is for Wordpress Multisite installation. It adds some fields on new blog form, including client name and FTP credentials. This plugin is used along with Simply Static plugin and requires it to work properly. Simply Static will generate a static version of the site, and this plugin can upload this static version to the client's hosting.

== Installation ==

1. Upload this zip file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
